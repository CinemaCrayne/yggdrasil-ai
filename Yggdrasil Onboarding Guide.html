Yggdrasil AI Onboarding: Using the LLM Plugin

What You Can Do

Use the chat interface to naturally store, retrieve, and reason with memory using Yggdrasil AI. Just speak to it—like this:

Add Memory

  • Goal: Remember that I want to write a paper on LLM ethics.
  • Insight: Note that early testing leads to better user retention.
  • Fact: Store that team syncs happen every Friday.
  • Reflection: I realized I get more done in the mornings.

Retrieve Memory

  • What are my current goals?
  • Summarize insights about UX design.
  • Do I have any reflections on productivity?

Update Memory

  • Update my goal to finish the paper by October 1.
  • Actually, our meetings moved to Tuesdays.

Example Interaction

You: Remember I want to build a GPT-powered calendar assistant.

Yggdrasil AI: Got it. I’ve saved your goal to build a GPT-powered calendar assistant.

You: What are my current goals?

Yggdrasil AI: Your goals:
1. Build a GPT-powered calendar assistant

You’re now ready to get started. Engage naturally—Yggdrasil AI will remember, reflect, and help you reason.
